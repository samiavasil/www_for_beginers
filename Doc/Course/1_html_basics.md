# 01. Основи на HTML

## Какво е HTML?
HTML (HyperText Markup Language) е основният език за създаване на уеб страници. Той описва структурата на съдържанието чрез т.нар. "тагoве".

## Структура на HTML документ
Всеки HTML документ започва с декларация `<!DOCTYPE html>`, след което следват основните тагове:
- `<html>`: кореновият елемент
- `<head>`: мета информация, заглавие, стилове
- `<body>`: видимото съдържание

## Основни тагове
- `<nav>`: навигационно меню
- `<main>`: основно съдържание
- `<section>`: отделни части
- `<footer>`: долна част

## Пример
```html
<!DOCTYPE html>
<html lang="bg">
  <head>
    <meta charset="UTF-8">
    <title>Моят първи сайт</title>
  </head>
  <body>
    <nav>Меню</nav>
    <main>Съдържание</main>
    <footer>Футър</footer>
  </body>
</html>
```
